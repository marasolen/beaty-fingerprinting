Beaty Biodiversity Museum: Fingerprints

For each row, there is a set of "cells" layed out using a Voronoi map. Each cell represents a group of specimens in the row. There are three attributes: size, aquatic to terrestrial, and flies vs doesn't fly. Each cell is a specific combination of these three attributes. The size of the cell represents how many specimens in the row match that combination of attributes, and the coloured, textured fills represents the attributes, with size in the centre, aquatic to terrestrial on the outside, and ability to fly in between.
